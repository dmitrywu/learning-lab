# Git basics

## Ментальная модель

Git хранит не просто файлы, а историю изменений проекта.

## Базовый цикл

1. Изменить файлы.
2. Проверить состояние через `git status`.
3. Добавить изменения через `git add`.
4. Создать коммит через `git commit`.
5. Отправить изменения на GitHub через `git push`.

## Команды

``bash
git status
git add .
git commit -m "Add git basics note"
git push
```
