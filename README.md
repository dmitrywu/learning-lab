# learning-lab

Учебный репозиторий для пути 0 → архитектор высоконагруженных систем.

## Основная траектория

1. Go
2. Backend
3. SQL / PostgreSQL
4. Linux / сети / ОС
5. Python для DevOps
6. Docker / CI/CD
7. Kubernetes / Terraform
8. Observability / SRE
9. System Design
10. Distributed Systems

## Правило обучения

Каждый этап должен давать проверяемый артефакт:
- код;
- заметку;
- тест;
- проект;
- design doc;
- разбор ошибки.
